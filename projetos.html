Geoapify Attractions Finder — Python + APIs

Aplicação interativa em Python que, a partir de coordenadas e raio de busca, consulta a API Geoapify para encontrar pontos de interesse (restaurantes, museus, hotéis etc.). Utiliza pandas para estruturar os dados, integra a API Restcountries para obter a moeda local de cada país e permite ordenação por nome ou distância. O resultado é exportado para um arquivo CSV pronto para análise.

Hangman Game com ASCII Art, Dicas e Salvamento

Jogo da forca completo em Python, executado no terminal. Inclui desenho da forca em ASCII, validação de entrada, suporte a acentos e cedilhas, sistema de dicas (gasta 5 pontos para revelar uma letra), pontuação acumulada salva em arquivo (save.txt), e menu interativo com opção de escolha entre listas de palavras com ou sem acentos. Código modularizado em main.py, game.py, ferramentas.py e wordslist.py.
